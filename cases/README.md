# Casi studio

La cartella è separata in due aree:

- `students/` → materiale da consegnare agli studenti (casi anonimi, didattici, completi)
- `instructor/` → guida riservata al docente con chiavi di lettura dei casi

## Accesso rapido
- Studenti: `cases/students/README.md`
- Docente: `cases/instructor/README.md`
