# Setup area privata docente

La cartella `private_instructor/` è ignorata da Git e serve per tenere soluzioni/materiali non visibili agli studenti.

Esempio percorso consigliato:
- `private_instructor/lezione1/SOLUZIONI_ISTRUTTORE.md`

Per iniziare velocemente:
```bash
mkdir -p private_instructor/lezione1
cp Lezioni/lezione1_pratica_bash/SOLUZIONI_ISTRUTTORE.template.md private_instructor/lezione1/SOLUZIONI_ISTRUTTORE.md
```
